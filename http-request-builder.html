HTTP 요청 빌더란?

URL, HTTP 메서드, 쿼리 파라미터, 헤더, 요청 본문을 시각적으로 설정하고 cURL, fetch(), axios 코드를 자동 생성합니다. API 문서 작성, 팀 협업, 학습 목적에 유용합니다.

자주 묻는 질문

실제로 API를 호출하나요?

HTTP 요청 빌더는 코드만 생성합니다. 실제 호출은 api-tester 도구를 사용하세요.

쿼리 파라미터는 자동으로 URL에 추가되나요?

네. Key-Value로 입력한 파라미터는 자동으로 인코딩되어 URL에 추가됩니다.

fetch() 코드를 바로 사용할 수 있나요?

모던 브라우저나 Node.js 18+ 환경에서 바로 사용 가능합니다.

Bearer Token은 어떻게 추가하나요?

헤더에 Authorization: Bearer {token}을 직접 입력하세요.

form-data 본문도 지원하나요?

현재는 JSON 본문만 지원합니다. multipart form-data는 추후 지원 예정입니다.

axios 코드도 생성되나요?

네. axios 탭에서 axios 스타일의 요청 코드를 확인할 수 있습니다.

URL 인코딩은 자동으로 처리되나요?

쿼리 파라미터 값은 encodeURIComponent로 자동 인코딩됩니다.

복수의 헤더를 추가할 수 있나요?

+ Add 버튼으로 원하는 만큼 헤더를 추가할 수 있습니다.
