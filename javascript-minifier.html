JavaScript 압축기

JavaScript 코드에서 주석, 공백, 줄바꿈을 제거해 파일 크기를 줄입니다. // 한줄 주석, /* */ 블록 주석 제거, 연속 공백 및 연산자 주변 공백 제거를 지원합니다. 프로덕션 배포 전 간단한 압축이 필요할 때 사용하세요.

자주 묻는 질문

JS 미니파이어란?

JavaScript 파일에서 주석, 공백, 줄바꿈을 제거해 파일 크기를 줄이는 도구입니다.

압축 후 동작이 바뀌나요?

기본 공백/주석 제거는 동작에 영향을 주지 않습니다.

변수명 단축은 지원하나요?

변수명 단축(맹글링)은 이 도구에서 지원하지 않습니다. Terser나 UglifyJS를 사용하세요.

문자열 안에 // 가 있으면?

정규식 기반 주석 제거 시 문자열 내 // 가 오탐될 수 있습니다. 복잡한 코드는 Terser 권장.

프로덕션 권장 도구는?

Terser, UglifyJS, esbuild, webpack 등을 권장합니다.

정규식 리터럴이 있으면?

정규식 내부 공백은 보존됩니다. 패턴 외부 공백만 제거됩니다.

TypeScript도 압축 가능한가요?

이 도구는 순수 JS만 지원합니다. TypeScript는 먼저 컴파일 후 사용하세요.

ES6+ 문법도 지원하나요?

예, 화살표 함수, 템플릿 리터럴, async/await 등 ES6+ 코드에서도 공백/주석 제거가 동작합니다.
