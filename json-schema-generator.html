JSON Schema 생성기

JSON 데이터를 입력하면 Draft-07 형식의 JSON Schema를 자동으로 생성합니다. required 배열, additionalProperties 제한, title/description 필드 포함 여부를 옵션으로 설정할 수 있습니다.

자주 묻는 질문

JSON Schema란 무엇인가요?

JSON 데이터의 구조와 유효성을 정의하는 표준입니다. API 문서, 폼 유효성 검사, 데이터 계약에 사용됩니다.

Draft-07과 다른 버전의 차이는?

Draft-07은 현재 가장 널리 사용되는 버전으로 if/then/else 조건부 스키마를 지원합니다.

additionalProperties: false가 무엇인가요?

스키마에 정의되지 않은 추가 속성을 허용하지 않는 설정입니다.

배열이 있는 JSON도 지원하나요?

예, 배열 내 객체의 구조도 분석하여 items 스키마를 자동 생성합니다.

중첩 객체도 스키마가 생성되나요?

예, 재귀적으로 중첩 객체의 스키마도 생성됩니다.

null 값은 어떻게 처리되나요?

null 타입으로 처리하여 type: ["string","null"]과 같이 nullable을 표현합니다.

생성된 스키마를 바로 사용할 수 있나요?

초안으로 사용하되, 실제 요구사항에 맞게 제약 조건(minLength, minimum, enum 등)을 추가하세요.

AJV, Joi 등과 호환되나요?

JSON Schema Draft-07은 AJV, Joi, Zod 등 대부분의 유효성 라이브러리와 호환됩니다.
