    SVG Cleaner

    Adobe Illustrator, Inkscape, Figma, Sketch 등의 벡터 편집 프로그램에서 내보낸 SVG 파일에는 웹에서 사용하기 위한 불필요한 에디터 전용 요소들이 포함됩니다. 이 도구는 이러한 요소들을 제거하여 웹에서 사용할 수 있는 깔끔한 SVG 파일을 만들어줍니다. Inkscape의 sodipodi/inkscape 네임스페이스, Illustrator의 i:/serif: 네임스페이스 등을 정리합니다.

    자주 묻는 질문

    SVG Optimizer는 어디로 갔나요?

    SVG Optimizer는 이 도구(SVG Cleaner)로 통합되었습니다. 에디터 전용 요소 제거와 파일 크기 최적화를 이제 하나의 도구에서 모두 처리합니다.

    Inkscape SVG를 정리하면 어떤 요소가 제거되나요?

    sodipodi:, inkscape: 네임스페이스의 모든 요소와 속성이 제거됩니다. 이는 Inkscape 편집 내부 정보로 렌더링에 영향을 주지 않습니다.

    정리 후 SVG가 제대로 표시되는지 어떻게 확인하나요?

    출력 코드를 브라우저에서 직접 열거나 SVG Viewer 도구를 사용해 미리보기를 확인하세요.

    Figma에서 내보낸 SVG도 정리가 되나요?

    예, Figma 전용 속성(figma:, data-figma-)이 제거됩니다.

    에디터 생성 ID 제거 옵션은 언제 사용하나요?

    SVG를 웹에 인라인으로 삽입할 때 Page_1, Layer_2 같은 자동 ID가 충돌할 수 있습니다. 이때 제거가 유용합니다.

    정리 후 파일 크기가 얼마나 줄어드나요?

    에디터에 따라 다르지만 Inkscape SVG는 메타데이터가 많아 10~40% 감소하는 경우가 많습니다.

    파일이 서버에 업로드되나요?

    아니요. 모든 처리가 브라우저에서 직접 이루어집니다.

    결과를 어떻게 사용하나요?

    복사 버튼으로 클립보드에 복사하거나, 다운로드 버튼으로 .svg 파일로 저장하세요.

    빈 그룹이나 DOCTYPE 선언도 제거되나요?

    네. 에디터 전용 네임스페이스뿐 아니라 빈 <g> 그룹, DOCTYPE 선언, visibility:hidden 요소 같은 불필요한 항목도 함께 정리해 파일 크기를 줄입니다.
