JavaScript 포매터

압축되거나 한 줄로 된 JavaScript 코드를 들여쓰기와 줄바꿈으로 보기 좋게 정렬합니다. 들여쓰기 크기(2칸, 4칸, 탭)와 중괄호 스타일(같은 줄, 다음 줄) 선택을 지원합니다. 소스코드 분석, 디버깅, 코드 리뷰에 유용합니다.

자주 묻는 질문

JS 비우티파이어란?

압축된 JavaScript를 들여쓰기와 줄바꿈으로 보기 좋게 정렬하는 도구입니다.

난독화된 코드도 해독되나요?

공백/줄바꿈 복원은 가능하지만 변수명 난독화는 복원할 수 없습니다.

들여쓰기는 몇 칸이 좋나요?

일반적으로 2칸 또는 4칸을 사용합니다. 프로젝트 스타일에 맞추세요.

Prettier와 차이는?

이 도구는 간단한 공백/줄바꿈 복원만 지원합니다. AST 기반 포맷은 Prettier를 사용하세요.

TypeScript 코드도 지원하나요?

순수 JS 포맷만 지원합니다. TypeScript는 TS 전용 포매터를 사용하세요.

문자열 내용이 바뀌지 않나요?

문자열 내부는 수정하지 않습니다. 따옴표 밖 공백만 조정됩니다.

중괄호 스타일 차이는?

같은 줄(K&R): function foo() {. 다음 줄(Allman): function foo() {.

ES6+ 문법도 지원하나요?

예, 화살표 함수, 구조분해, 템플릿 리터럴 등 포맷이 가능합니다.
