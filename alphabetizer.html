알파벳 순 정렬 도구 사용법

텍스트의 각 줄을 알파벳 오름차순(A→Z) 또는 내림차순(Z→A)으로 정렬합니다. 대소문자 구분, 중복 제거, 앞뒤 공백 제거 옵션을 지원하며 한국어 가나다순 정렬도 가능합니다.

한국어도 정렬되나요?

네. localeCompare를 사용하여 한국어 가나다순 정렬도 지원합니다. 영어뿐 아니라 중국어, 일본어 등 유니코드 텍스트도 정렬 가능합니다.

중복 줄을 함께 제거할 수 있나요?

네. "중복 제거" 옵션을 체크하면 정렬과 동시에 중복 줄을 제거합니다. 대소문자 구분 옵션과 연동되어 Hello와 hello를 같은 것으로 처리할지 선택할 수 있습니다.

숫자가 포함된 텍스트는 어떻게 정렬되나요?

기본 알파벳 정렬에서는 숫자를 문자로 처리하여 "10"이 "2"보다 앞에 옵니다. 숫자를 값으로 비교하려면 Text Line Sorter의 숫자순 정렬 기능을 사용하세요.

정렬 결과를 어떻게 저장하나요?

복사 버튼을 눌러 결과를 클립보드에 복사한 뒤 원하는 곳에 붙여넣으세요. 별도의 저장 기능은 없으며 모든 처리는 브라우저 내에서만 이루어집니다.

빈 줄도 정렬 결과에 포함되나요?

"빈 줄 제거" 옵션을 선택하면 정렬 결과에서 빈 줄이 제거됩니다. 선택하지 않으면 빈 줄도 하나의 항목으로 처리되어 정렬 결과에 포함됩니다.

대소문자를 구분해서 정렬할 수 있나요?

네. 대소문자 구분(Case-sensitive) 옵션을 선택하면 대문자가 소문자보다 앞에 오는 ASCII 순서로 정렬됩니다. 옵션 해제 시 대소문자 없이 알파벳순 정렬됩니다.

역순(Z→A) 정렬도 가능한가요?

가능합니다. 정렬 방향 옵션에서 내림차순(Descending)을 선택하면 역순으로 정렬됩니다. 날짜나 숫자가 포함된 목록을 최신·최대 기준으로 정렬할 때 유용합니다.

일본어나 중국어 텍스트도 정렬할 수 있나요?

유니코드 코드 포인트 순으로 정렬됩니다. 일본어(히라가나/가타카나)와 중국어(한자)는 언어별 사전 순이 아닌 유니코드 순서로 정렬됩니다. 정확한 언어별 정렬이 필요하면 해당 언어의 collation 라이브러리를 사용하세요.
