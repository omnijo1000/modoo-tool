CSV to JSON 변환기란?

CSV 첫 행을 키(key)로 사용해 JSON 객체 배열로 변환합니다. 쉼표·세미콜론·탭 등 구분자를 자동 감지하며, 따옴표로 감싼 필드·줄바꿈 포함 값·BOM 문자도 처리합니다. API 연동·데이터 마이그레이션·스프레드시트 데이터 처리에 활용하세요.

자주 묻는 질문 (FAQ)

구분자(delimiter)를 자동으로 감지하나요?

네. 쉼표(,)·세미콜론(;)·탭(\t)을 자동으로 감지합니다. 필요 시 수동으로 구분자를 선택할 수도 있습니다.

헤더(첫 행)가 없는 CSV는 어떻게 처리하나요?

이 변환기는 첫 행을 키로 사용합니다. 헤더가 없는 경우 첫 행 값이 키가 되므로, 먼저 헤더 행을 추가하거나 결과에서 수동으로 수정하세요.

값 안에 쉼표나 줄바꿈이 있으면 어떻게 되나요?

RFC 4180 표준대로 따옴표(")로 감싸면 내부 쉼표·줄바꿈을 올바르게 처리합니다. 예: "Seoul, Korea"

JSON 들여쓰기 설정이 가능한가요?

네. 0·2·4 스페이스 및 탭 들여쓰기 중 선택할 수 있습니다. 0을 선택하면 최소화된(minified) JSON이 출력됩니다.

변환 결과를 파일로 저장할 수 있나요?

네. 변환 후 "다운로드 .json" 버튼을 클릭하면 JSON 파일로 저장됩니다.

변환 후 JSON에서 숫자와 불리언 타입이 자동 변환되나요?

네. "123" 같은 숫자 문자열과 "true"/"false" 불리언 문자열은 JSON에서 각각 숫자 및 불리언 타입으로 자동 변환됩니다.

첫 행이 헤더가 아닌 데이터라면 어떻게 해야 하나요?

헤더 없음 옵션을 선택하면 첫 행도 데이터로 처리됩니다. 변환 결과는 col1, col2처럼 자동 생성된 키를 사용합니다.

공백 문자가 포함된 CSV 값은 어떻게 처리되나요?

셀 앞뒤의 공백은 그대로 보존됩니다. 필요한 경우 변환 후 각 값에 trim 처리를 별도로 적용하세요.
