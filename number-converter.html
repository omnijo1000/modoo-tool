진수 변환기란?

2진수·8진수·10진수·16진수를 실시간으로 상호 변환하며, 32비트 이진 표현도 함께 표시합니다. 개발·임베디드·네트워킹에서 자주 필요한 진수 변환을 빠르게 처리할 수 있습니다.

각 진수 표기법은 어떻게 구분하나요?

2진수(Binary): 0과 1만 사용, 0b 접두사. 8진수(Octal): 0–7, 0o 접두사. 10진수(Decimal): 0–9, 일반 숫자. 16진수(Hex): 0–9와 A–F, 0x 접두사. 예: 255 = 0xFF = 0b11111111 = 0o377.

왜 16진수를 많이 사용하나요?

16진수 1자리가 정확히 4비트(=1 니블)에 대응하여, 2자리가 1바이트를 표현합니다. 색상 코드(#FF5733), 메모리 주소(0x7FFF0000), 비트마스크 등에서 사람이 읽기 쉬운 이진 표현으로 사용됩니다.

음수는 어떻게 표현하나요?

컴퓨터는 2의 보수(Two's Complement) 방식을 사용합니다. 32비트 부호 있는 정수에서 최상위 비트(MSB)가 1이면 음수입니다. 예: 0xFFFFFFFF = -1, 0x80000000 = -2,147,483,648.

16진수를 10진수로 수동 변환하는 방법은?

각 자리에 16의 거듭제곱을 곱해 더합니다. 예: 0x1A = 1×16¹ + 10×16⁰ = 16 + 10 = 26. A=10, B=11, C=12, D=13, E=14, F=15. 또는 이 계산기에 입력하면 즉시 변환됩니다.

입력할 수 있는 수의 범위는 얼마인가요?

이 계산기는 32비트 부호 없는 정수(0 ~ 4,294,967,295 / 0x00000000 ~ 0xFFFFFFFF) 범위를 기준으로 동작합니다. 범위를 벗어나는 값은 잘려나거나 오류가 발생할 수 있습니다.

부동소수점(float) 숫자도 변환할 수 있나요?

이 도구는 정수 진수 변환에 최적화되어 있습니다. 부동소수점은 지수부와 가수부를 IEEE 754 방식으로 별도 처리해야 하므로, 소수점이 있는 수의 2진수 표현이 필요하다면 IEEE 754 변환 전용 도구를 사용하세요.

매우 큰 수(64비트 이상)를 변환할 수 있나요?

JavaScript의 Number 타입은 53비트 정수를 안전하게 표현합니다(Number.MAX_SAFE_INTEGER = 2^53 - 1). 그 이상의 정수는 BigInt를 지원하는 변환 도구나 Python 같은 임의 정밀도 언어를 사용하세요.

색상 코드(#FF5733)를 RGB로 변환할 수 있나요?

색상 코드는 16진수로 표현됩니다. 이 도구에서 "FF5733"을 16진수로 입력하면 10진수 16736051로 변환됩니다. R=FF(255), G=57(87), B=33(51)은 각 쌍을 개별 변환해야 합니다. 색상 변환 전용 도구가 더 편리합니다.
