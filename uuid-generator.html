UUID v4는 122비트의 암호학적 난수로 생성됩니다. 충돌 확률은 약 1/2¹²²로 사실상 전 세계에서 고유합니다. 데이터베이스 기본 키, 세션 토큰, API 요청 ID 등에 활용됩니다.

UUID란?

UUID(Universally Unique Identifier)는 128비트 길이의 식별자로, 전 세계 어디서나 고유한 값을 보장합니다. v4는 완전 랜덤 방식으로 생성되며, RFC 4122 표준을 따릅니다. 형식은 8-4-4-4-12 자리의 16진수입니다.
