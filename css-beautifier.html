CSS 포매터

압축된 CSS 코드에 들여쓰기와 줄바꿈을 추가해 읽기 좋게 정렬합니다. 들여쓰기 크기 선택, 선택자별 줄바꿈, 속성 알파벳 정렬을 지원합니다. 미디어 쿼리, @keyframes, :root 변수 등 모든 CSS 구문을 포맷합니다.

자주 묻는 질문

CSS 포매터란?

압축된 CSS에 들여쓰기와 줄바꿈을 추가해 가독성을 높이는 도구입니다.

미디어 쿼리도 포맷되나요?

예, @media 블록과 중첩 규칙도 들여쓰기가 적용됩니다.

속성 정렬이란?

CSS 속성을 알파벳 순서로 정렬해 일관성을 높입니다.

SCSS도 포맷 가능한가요?

순수 CSS만 지원합니다. SCSS는 전용 포매터를 사용하세요.

Prettier와 차이는?

간단한 공백/줄바꿈 복원만 지원합니다. 심화 포맷은 Prettier 사용 권장.

CSS 변수도 포맷되나요?

예, --variable 형태의 CSS 사용자 속성도 정상 포맷됩니다.

@keyframes도 지원하나요?

예, @keyframes 블록도 들여쓰기가 적용됩니다.

선택자에 쉼표가 있으면?

선택자별 줄바꿈 옵션 활성화 시 쉼표 다음에 줄바꿈이 추가됩니다.
