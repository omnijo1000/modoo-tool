SSH 키 생성기란?

SSH 공개키 인증을 위한 키 쌍 생성 명령어를 자동으로 만들어주는 도구입니다. Ed25519(권장), RSA 2048/4096비트를 지원합니다. 브라우저에서 OpenSSH 포맷의 키를 직접 생성하는 것은 기술적 제한이 있어, 표준 ssh-keygen 명령어를 생성해 드립니다.

자주 묻는 질문

Ed25519와 RSA 중 무엇을 선택해야 하나요?

Ed25519를 권장합니다. 현대적이고 빠르며 보안성이 높습니다. 레거시 시스템 호환이 필요한 경우만 RSA를 사용하세요.

SSH 키는 어디에 저장되나요?

기본적으로 ~/.ssh/ 디렉토리에 저장됩니다. 개인키는 id_ed25519, 공개키는 id_ed25519.pub 파일입니다.

공개키를 서버에 등록하는 방법은?

ssh-copy-id user@host 명령어를 사용하거나, 공개키 내용을 서버의 ~/.ssh/authorized_keys 파일에 추가합니다.

패스프레이즈를 설정해야 하나요?

보안을 위해 권장합니다. ssh-agent를 사용하면 매번 입력하지 않아도 됩니다.

기존 키가 있으면 덮어씌워지나요?

ssh-keygen은 기존 키가 있으면 덮어쓸지 묻습니다. 별도 파일 경로를 지정하면 기존 키를 유지할 수 있습니다.

GitHub에 SSH 키를 등록하는 방법은?

GitHub 설정 → SSH and GPG keys → New SSH key에서 공개키(.pub 내용)를 붙여넣기 합니다.

Windows에서 SSH 키를 생성하려면?

Git Bash, WSL, 또는 Windows 10 이상의 내장 OpenSSH를 통해 동일한 ssh-keygen 명령어를 사용할 수 있습니다.

RSA 2048비트는 충분히 안전한가요?

현재까지 2048비트는 충분한 보안을 제공합니다. 더 높은 보안이 필요하다면 4096비트나 Ed25519를 선택하세요.
