Text Line Sorter 사용법

텍스트의 줄을 알파벳순(A→Z, Z→A), 길이순, 숫자순, 무작위로 정렬합니다. 중복 제거, 빈 줄 제거, 앞뒤 공백 제거 옵션도 제공하여 목록 정리 작업을 빠르게 처리할 수 있습니다.

숫자순 정렬은 어떻게 작동하나요?

숫자순 정렬은 각 줄의 선행 숫자를 기준으로 정렬합니다. 예를 들어 "10번", "2번", "1번"이 있을 경우 텍스트 정렬(A→Z)은 1, 10, 2 순서가 되지만, 숫자순은 1, 2, 10 순서로 올바르게 정렬합니다.

중복 제거는 대소문자를 구분하나요?

대소문자 구분 옵션에 따라 달라집니다. 대소문자 구분을 끄면 "Hello"와 "hello"를 같은 줄로 처리하여 하나만 남깁니다. 켜면 두 줄이 다른 것으로 간주됩니다.

줄이 아닌 단어 단위로 정렬할 수 있나요?

이 도구는 줄(line) 단위 정렬만 지원합니다. 단어 단위 정렬이 필요하다면 텍스트를 줄바꿈으로 분리한 후 정렬하는 방법을 사용하세요. 한 줄에 한 단어씩 입력하면 단어 정렬과 동일한 결과를 얻을 수 있습니다.

정렬 결과를 어디에 활용할 수 있나요?

이메일 목록 정렬, URL 중복 제거, CSV 데이터 정리, 코드 import 구문 정렬, 단어 목록 가나다순 정리 등에 활용할 수 있습니다. 개발자와 비개발자 모두 자주 사용하는 텍스트 유틸리티입니다.

처리 가능한 줄 수 제한이 있나요?

특별한 제한은 없으며 브라우저 메모리 허용 범위 내에서 처리됩니다. 수만 줄 이상의 대용량 텍스트도 처리 가능하지만, 매우 큰 경우 브라우저가 느려질 수 있습니다.

자연 정렬(natural sort)이란 무엇인가요?

자연 정렬은 숫자가 포함된 텍스트를 사람이 읽는 방식으로 정렬합니다. 예: item1, item2, item10 순서. 일반 알파벳 정렬로는 item1, item10, item2 순으로 나오지만 자연 정렬은 1→2→10 순을 유지합니다.

정렬 후 중복을 제거하는 옵션이 있나요?

네. 중복 제거 옵션을 함께 사용하면 정렬과 동시에 중복 줄을 제거합니다. 고유한 항목만 알파벳순으로 보고 싶을 때 한 번에 처리할 수 있습니다.

한국어 텍스트는 어떤 순서로 정렬되나요?

한국어는 유니코드 코드 포인트 순으로 정렬됩니다. 가나다 순과 거의 일치하지만, 완전한 한국어 사전 정렬이 필요하면 ko-KR 로케일 기반의 정렬 도구를 사용하세요.
