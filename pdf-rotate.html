PDF 페이지 회전

PDF 페이지를 90°·180°·270° 회전합니다. 전체 페이지 일괄 회전 또는 특정 페이지만 선택하여 회전할 수 있습니다. 스캔된 PDF의 방향이 잘못된 경우 손쉽게 수정할 수 있습니다.

자주 묻는 질문

특정 페이지만 회전할 수 있나요?

예, 페이지 범위를 입력하세요. 예: 1,3,5-8

파일이 서버로 전송되나요?

아니요. 브라우저 내에서만 처리됩니다.

PDF 내용이 변경되나요?

아니요. 페이지 방향만 변경됩니다.

270° 회전은 왼쪽 회전인가요?

예, 270° 오른쪽 회전은 90° 왼쪽 회전과 동일합니다.

대용량 PDF도 처리할 수 있나요?

브라우저 메모리 한도 내에서 처리됩니다. 수백MB 이상은 느릴 수 있습니다.

암호화된 PDF도 회전할 수 있나요?

암호화되지 않은 PDF만 처리됩니다. 먼저 비밀번호를 제거하세요.

회전 후 파일 품질이 달라지나요?

아니요. 품질 변화 없이 방향만 수정됩니다.

모바일에서도 사용할 수 있나요?

예, 모던 모바일 브라우저에서 동작합니다.
