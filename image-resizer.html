Image Resizer 완벽 가이드

이미지 크기를 정확하게 조절해야 할 때 가장 빠르고 안전한 도구입니다. Canvas API를 사용해 브라우저 내에서만 처리되므로 파일이 서버로 전송되지 않습니다. 픽셀(px) 단위로 정확한 크기를 지정하거나 원본 대비 비율(%)로 확대·축소할 수 있으며, 비율 잠금(Aspect Ratio Lock) 기능으로 가로·세로 비율이 자동 유지됩니다. SNS 프로필 사진, 웹사이트 배너, 이메일 인라인 이미지 등 플랫폼별 권장 크기에 맞게 빠르게 조정하세요. Google PageSpeed의 "적절한 이미지 크기 제공" 경고도 이 도구로 해결할 수 있습니다.

활용 예시

  • Instagram 프로필 사진 (1080×1080px) 크기로 정확하게 자르기
  • 웹사이트 배너 이미지를 디스플레이 크기로 축소 (불필요한 픽셀 제거)
  • 이메일 인라인 이미지를 600px 너비로 조정
  • 고해상도 제품 사진을 썸네일 크기(300×300px)로 생성
  • 인쇄용 이미지를 300DPI 기준 픽셀 크기로 변환
  • 동영상 썸네일(YouTube: 1280×720, Reels: 1080×1920) 크기 맞추기

자주 묻는 질문

Q. 픽셀 모드와 비율 모드의 차이는 무엇인가요?

픽셀 모드: 정확한 출력 크기를 지정 (예: 1920×1080). 비율 모드: 원본 크기 대비 백분율로 조절 (예: 50% = 원본의 절반 크기).

Q. 비율 잠금은 어떻게 작동하나요?

비율 잠금 활성화 시 가로 또는 세로 중 하나를 변경하면 나머지가 원본 비율에 맞게 자동 계산됩니다. 이미지가 찌그러지지 않게 유지할 때 필수 기능입니다.

Q. 이미지 크기를 키우면(업스케일) 화질이 좋아지나요?

아니요. 업스케일(픽셀 수 증가)은 AI 초해상도 기술 없이는 실제 디테일이 추가되지 않으며, 단순 보간으로 약간 흐릿해질 수 있습니다. 원본보다 크게 만들기보다는 축소 방향으로 사용하는 것이 좋습니다.

Q. 파일이 서버로 전송되나요?

아니요. Canvas API 기반으로 브라우저 내에서만 처리됩니다. 오프라인에서도 동작하며 개인 사진도 안전합니다.

Q. SNS 프로필 사진 권장 크기가 얼마인가요?

Instagram: 1080×1080px (정사각형), Facebook: 170×170px (프로필), Twitter/X: 400×400px, YouTube: 800×800px. 각 플랫폼이 자동으로 원형 마스크를 적용하므로 정사각형으로 리사이즈 후 업로드.

Q. 품질(Quality) 설정은 어떤 역할을 하나요?

출력 형식이 JPEG 또는 WebP일 때 압축 품질을 조절합니다. PNG는 무손실이므로 품질 설정이 적용되지 않습니다. 웹 표시용: 75~80 권장.

Q. 리사이즈 후 파일 형식을 변경할 수 있나요?

가능합니다. 출력 형식(JPEG/WebP/PNG)을 선택해 리사이즈와 형식 변환을 동시에 수행할 수 있습니다.

Q. 투명 배경(PNG)을 리사이즈하면 투명도가 유지되나요?

PNG 출력 또는 WebP 출력 선택 시 투명도 유지됩니다. JPEG 출력 선택 시 투명 영역이 흰색으로 채워집니다.

Q. Google PageSpeed "적절한 이미지 크기 제공" 경고는 어떻게 해결하나요?

PageSpeed가 감지한 실제 표시 크기로 이미지를 리사이즈하세요. 2배 해상도(Retina) 화면을 위해 표시 크기의 2배 픽셀로 제공하는 것도 허용됩니다.

Q. 처리할 수 있는 이미지 크기 제한이 있나요?

브라우저 Canvas 메모리 한도 내에서 처리됩니다. 4K(3840×2160) 이하는 대부분의 기기에서 정상 동작합니다. 8K 이상 초고해상도 이미지는 메모리 부족으로 실패할 수 있습니다.

Q. 리사이즈된 이미지의 파일명은 어떻게 되나요?

원본 파일명에 '_resized' 접미사가 추가되고 선택한 출력 형식의 확장자로 저장됩니다. (예: photo_resized.webp)

Q. 인쇄용 이미지는 어떤 크기로 설정해야 하나요?

인쇄 품질은 DPI(인치당 픽셀 수)로 결정됩니다. A4 고품질 인쇄(300DPI): 2480×3508px. A4 일반 인쇄(150DPI): 1240×1754px. 화면 확인용(72DPI)은 595×842px.
