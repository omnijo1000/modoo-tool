PDF Merge 완벽 가이드

PDF 파일을 합치는 가장 쉽고 안전한 방법입니다. pdf-lib 라이브러리를 사용해 브라우저 내에서 직접 처리하므로 파일이 외부 서버로 전송되지 않아 기밀 문서도 안전하게 병합할 수 있습니다. 계약서, 보고서, 이력서 첨부 자료 등 여러 PDF 파일을 드래그 앤 드롭으로 추가하고 순서를 자유롭게 조정한 후 한 번의 클릭으로 합칩니다. 별도의 소프트웨어 설치가 필요 없으며 Windows, Mac, 모바일 브라우저 모두에서 사용 가능합니다. 합쳐진 파일은 원본 PDF의 페이지 내용과 레이아웃을 그대로 유지합니다.

활용 예시

  • 계약서 본문 + 첨부 서류 + 서명란을 하나의 PDF로 통합
  • 여러 페이지로 나뉜 스캔 문서를 단일 파일로 병합
  • 월간 보고서 표지 + 본문 + 데이터 부록을 순서대로 합치기
  • 이력서 + 자기소개서 + 포트폴리오를 하나의 지원 패키지로
  • 여러 파트로 나뉜 교재/매뉴얼 PDF를 하나로 통합
  • 여러 영수증·청구서를 하나의 파일로 묶어 제출

자주 묻는 질문

Q. 합칠 수 있는 PDF 파일 수나 크기 제한이 있나요?

브라우저 메모리 한도 내에서 처리됩니다. 파일 수 제한은 없으며 일반적으로 총 100MB 미만에서 원활하게 작동합니다. 대용량 스캔 PDF는 파일당 50MB 이하 권장.

Q. 파일 순서를 변경할 수 있나요?

파일 목록에서 드래그 앤 드롭으로 순서를 자유롭게 변경할 수 있습니다. 합치기 전에 원하는 순서로 정렬하세요.

Q. 파일이 서버로 전송되나요?

아니요. pdf-lib 라이브러리를 사용해 브라우저 로컬에서만 처리됩니다. 기밀 계약서, 개인 문서도 안전하게 사용 가능합니다.

Q. 암호화된(비밀번호 보호) PDF도 합칠 수 있나요?

비밀번호로 잠긴 PDF는 합칠 수 없습니다. 먼저 PDF 잠금 해제 도구를 사용해 보안을 해제한 후 병합하세요.

Q. 합쳐진 파일에서 북마크(목차)는 유지되나요?

현재 버전은 각 PDF의 페이지 내용을 병합하며, 기존 북마크/아웃라인은 병합 파일에 포함되지 않습니다.

Q. 스캔된 이미지 PDF도 합칠 수 있나요?

가능합니다. 스캔 PDF는 이미지 기반이므로 텍스트 인식 없이 페이지 이미지 그대로 병합됩니다.

Q. 모바일(스마트폰)에서도 사용할 수 있나요?

Chrome, Safari 등 현대적인 모바일 브라우저에서 모두 동작합니다. 파일 선택은 디바이스 파일 앱을 통해 접근할 수 있습니다.

Q. 합친 PDF의 페이지 크기나 레이아웃이 변경되나요?

아니요. 각 PDF 페이지의 원본 크기와 레이아웃이 그대로 유지됩니다. A4와 Letter 혼합도 각각의 원본 크기를 유지한 채 병합됩니다.

Q. 특정 페이지만 선택해서 합칠 수 있나요?

현재 버전은 파일 전체를 병합합니다. 특정 페이지만 선택하려면 PDF 분할/페이지 추출 도구를 먼저 사용해 필요한 페이지만 분리한 후 합치세요.

Q. 합친 파일의 이름을 지정할 수 있나요?

다운로드 시 브라우저 기본 저장 대화상자를 통해 파일명을 변경할 수 있습니다. 기본값은 'merged.pdf'입니다.

Q. PDF/A 또는 PDF/X 형식도 지원되나요?

일반 PDF는 모두 지원됩니다. PDF/A나 PDF/X 등 특수 규격 파일은 병합 후 해당 규격 준수 여부가 변경될 수 있습니다.

Q. 오피스(Word, PPT)를 PDF로 변환 후 합칠 수 있나요?

먼저 Office 파일을 PDF로 변환한 후 이 도구로 합치면 됩니다. Word/PPT → PDF 변환은 Microsoft Office의 '다른 이름으로 저장 → PDF' 또는 Google Docs의 다운로드 기능을 사용하세요.
