ASCII 코드표

#문자이름DECHEXOCTBINHTML

* 클릭 시 해당 문자 또는 10진수 값을 클립보드에 복사합니다.

ASCII 코드표란?

0~127번의 전체 ASCII 문자 코드표입니다. 10진수, 16진수, 8진수, 2진수, HTML 엔티티를 한눈에 확인하고 클릭하여 복사할 수 있습니다. 제어 문자, 숫자, 알파벳, 특수문자 필터를 지원합니다.

자주 묻는 질문

'A'는 ASCII 몇 번인가요?

대문자 'A'는 65번(10진수), 41(16진수)입니다. 소문자 'a'는 97번입니다.

제어 문자란?

0~31번 출력 불가능 문자입니다. LF(10=줄바꿈), CR(13=캐리지리턴), TAB(9), NULL(0)이 포함됩니다.

DEL(127)은 무엇인가요?

Delete 문자로, 원래 천공 테이프에서 한 문자를 삭제하는 데 사용했습니다.

Space는 ASCII 몇 번인가요?

공백(Space)은 32번입니다. HTML 엔티티는 &#32;입니다.

ASCII를 어디서 활용하나요?

C/C++ 문자 처리, 네트워크 프로토콜, 파일 형식 파싱 등에 활용합니다.

8진수(OCT)는 어디서 쓰나요?

Unix/Linux 파일 권한(chmod 755), C 언어 이스케이프 시퀀스(\007)에 사용합니다.

Extended ASCII(128~255)는 어디에 있나요?

이 표는 표준 ASCII(0~127)만 다룹니다. Extended ASCII는 플랫폼마다 다릅니다.

BEL(7)이란?

Bell 문자로, 원래 터미널에서 경고음을 내는 데 사용했습니다.
