Cron 표현식이란?

Cron은 Unix 계열 시스템에서 특정 시간에 자동으로 작업을 실행하는 스케줄러입니다. "분 시 일 월 요일" 5개 필드로 구성되며, Spring @Scheduled, Jenkins, GitHub Actions 등에서 사용합니다.

Spring @Scheduled 사용법

Spring Boot에서는 @Scheduled(cron = "0 0 9 * * MON-FRI") 형태로 사용합니다. Spring은 초(second) 필드가 추가되어 6개 필드를 사용하므로 맨 앞에 초(0)를 추가해야 합니다.

표준 cron(5필드) 기준 · Spring @Scheduled는 앞에 초(0~59) 필드 추가 필요
