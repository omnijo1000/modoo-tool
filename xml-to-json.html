XML to JSON 변환기

XML 데이터를 JSON으로 즉시 변환합니다. XML 속성(@), 텍스트 콘텐츠(#text), CDATA, 중첩 요소를 모두 처리합니다. RSS 피드, SOAP API, SVG, 레거시 시스템의 XML 데이터를 현대 API에 맞는 JSON으로 변환할 때 활용하세요.

자주 묻는 질문

XML 속성은 JSON에서 어떻게 표현되나요?

속성은 @속성명 키로 표현됩니다. <tag id="1"> → {"tag":{"@id":"1"}}.

텍스트 콘텐츠는?

텍스트는 #text 키로 표현됩니다. <name>Alice</name> → {"name":"Alice"} (자식 없는 경우 단순 문자열).

같은 태그가 여러 개면?

같은 이름의 요소가 여러 개면 배열로 처리됩니다.

XML 네임스페이스는?

xmlns 선언과 접두사는 속성으로 처리되거나 태그명에 포함됩니다.

CDATA는 어떻게 처리되나요?

CDATA 섹션의 내용은 텍스트로 추출됩니다.

RSS/Atom 피드도 변환 가능한가요?

네, RSS/Atom 피드는 표준 XML이므로 변환 가능합니다. 결과 구조는 피드 형식에 따라 다릅니다.

XML 선언(<?xml ...?>)은?

XML 선언은 변환 결과에서 제외되고 루트 요소부터 처리됩니다.

대용량 XML도 처리 가능한가요?

브라우저 DOMParser를 사용하므로 수 MB 이상의 XML은 성능이 저하될 수 있습니다.
