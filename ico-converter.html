ICO 변환기

PNG/JPG 이미지를 ICO 형식으로 변환합니다. 16×16, 32×32, 48×48 등 여러 크기를 선택하여 하나의 ICO 파일에 포함할 수 있습니다. 웹사이트 파비콘, Windows 아이콘 제작에 사용할 수 있습니다.

자주 묻는 질문

ICO와 PNG 파비콘의 차이는?

ICO는 여러 크기를 하나의 파일에 포함할 수 있어 구형 브라우저와 Windows에서 호환성이 좋습니다.

어떤 크기를 선택해야 하나요?

파비콘용: 16, 32, 48px. Windows 아이콘: 32, 48, 256px를 추천합니다.

투명도(알파채널)가 유지되나요?

PNG 투명도는 ICO 파일에서도 유지됩니다.

파일이 서버로 전송되나요?

아니요. 모두 브라우저 로컬에서 처리됩니다.

ICO 파일을 만들려면 원본이 정사각형이어야 하나요?

아니요, 직사각형 이미지도 사용할 수 있으며 자동으로 크롭됩니다.

ICO 파일을 파비콘으로 사용하려면?

파일명을 favicon.ico로 저장하고 웹 루트에 배치하세요.

256×256을 포함해야 하나요?

Windows 바로가기 아이콘 등 대형 아이콘이 필요한 경우 포함하세요.

생성된 ICO를 바로 미리보기할 수 있나요?

각 크기별 PNG 미리보기를 제공합니다.
