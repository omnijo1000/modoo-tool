PDF 크기 분석기

PDF 파일의 전체 크기, 페이지 수, 각 페이지 크기(A4, Letter 등), 메타데이터(제목, 작성자, 작성 날짜 등)를 분석합니다. PDF.js 라이브러리를 사용하여 브라우저에서 직접 처리하므로 파일이 서버에 업로드되지 않습니다.

자주 묻는 질문

PDF 페이지 크기는 어떻게 표시되나요?

각 페이지의 너비와 높이가 포인트(pt) 및 mm 단위로 표시되며, A4, Letter 등의 표준 크기와 비교합니다.

파일이 서버에 업로드되나요?

아니요. PDF.js를 사용하여 브라우저에서 직접 처리됩니다.

메타데이터가 없는 PDF도 분석 가능한가요?

예, 메타데이터가 없어도 파일 크기, 페이지 수, 페이지 크기는 확인할 수 있습니다.

PDF 크기를 줄이려면?

PDF Compressor 도구를 사용하거나, Acrobat에서 PDF 최적화 기능을 활용하세요.

암호화된 PDF도 분석할 수 있나요?

아니요. 먼저 PDF 잠금 해제 도구로 잠금을 해제해야 합니다.

여러 PDF를 한 번에 분석할 수 있나요?

현재 한 번에 하나의 PDF만 분석할 수 있습니다.

PDF 버전 정보는 무엇을 의미하나요?

PDF 형식 버전으로 1.0~2.0까지 있습니다. 최신 기능(암호화, 투명도 등)은 높은 버전이 필요합니다.

인쇄 시 페이지 크기가 다르게 나오는 이유는?

PDF 페이지 크기와 프린터 용지 크기가 다르면 인쇄 시 크기 조정이 발생합니다. 인쇄 설정에서 "실제 크기"로 설정하세요.
