Open Graph 미리보기

HTML 소스에서 OG 태그를 파싱하여 소셜 미디어 공유 화면을 미리 확인합니다

HTML 소스
직접 입력
og:title
og:url
og:description
og:image

오픈그래프 미리보기

HTML 소스 코드에서 Open Graph 메타 태그를 파싱하고 Facebook, Twitter, LinkedIn 공유 시 미리보기를 확인합니다. og:title, og:description, og:image, og:url 등을 검사하고 빠진 태그를 알려드립니다.

자주 묻는 질문

Open Graph 태그는 무엇인가요?

Facebook이 개발한 프로토콜로, 소셜 미디어에서 페이지가 공유될 때 제목, 설명, 이미지를 제어합니다.

og:image 크기 권장 사항은?

1200×630px를 권장하며, Facebook은 최소 600×315px 이상이어야 표시합니다.

Twitter도 OG 태그를 사용하나요?

Twitter는 자체 twitter: 태그를 우선하지만, 없으면 og: 태그를 사용합니다.

OG 태그를 업데이트해도 소셜 미디어에서 반영이 안 돼요

소셜 플랫폼은 OG 태그를 캐시합니다. Facebook Sharing Debugger로 캐시를 초기화할 수 있습니다.

og:type에는 어떤 값이 있나요?

website, article, book, profile, music, video 등이 있습니다. 기본값은 website입니다.

og:site_name은 무엇인가요?

소셜 카드에 표시되는 사이트 이름입니다. 브랜드명을 입력하세요.

여러 og:image 태그를 사용할 수 있나요?

가능하지만 대부분의 소셜 미디어는 첫 번째 이미지만 사용합니다.

og:locale은 무엇인가요?

콘텐츠의 언어와 지역을 지정합니다. 예: ko_KR, en_US, zh_CN.
