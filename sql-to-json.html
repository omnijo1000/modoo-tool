SQL INSERT 또는 CSV → JSON 배열 변환

SQL 입력
JSON 출력

SQL to JSON 변환기

SQL INSERT 문 또는 CSV 형식 데이터를 JSON 배열로 변환합니다. API 응답 시뮬레이션, 테스트 데이터 생성, 데이터 마이그레이션 등 다양한 용도로 활용합니다. INSERT INTO 구문의 컬럼명과 VALUES를 자동 파싱하고, CSV의 첫 번째 행을 헤더로 인식합니다.

자주 묻는 질문

여러 VALUES가 있는 INSERT도 처리되나요?

네. VALUES (a,b,c),(d,e,f); 와 같이 다중 행 INSERT를 지원합니다.

NULL 값은 어떻게 처리되나요?

SQL NULL은 JSON null로 변환됩니다.

숫자와 문자열을 자동으로 인식하나요?

따옴표 없는 숫자는 JSON number 타입으로, 따옴표로 감싼 값은 string 타입으로 처리합니다.

CSV에서 첫 번째 행이 헤더인가요?

네. CSV 모드에서는 첫 번째 행을 키 이름으로 사용합니다.

서버로 데이터가 전송되나요?

아니요. 모든 변환은 브라우저에서 이루어집니다.

결과 JSON은 pretty-printed인가요?

네. 2 스페이스 들여쓰기가 적용된 읽기 쉬운 형식으로 출력됩니다.

특수문자가 포함된 문자열도 처리되나요?

작은따옴표 이스케이프('')나 특수문자는 최대한 보존합니다.

JSON 이외 형식도 지원할 예정인가요?

현재 JSON만 지원합니다. 다른 도구를 이용해 추가 변환하세요.
