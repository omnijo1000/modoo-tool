CSS 압축기

CSS 코드에서 주석, 공백, 불필요한 문자를 제거해 파일 크기를 줄입니다. 마지막 세미콜론 제거, 0값 단위 제거, 색상 코드 단축 등 다양한 최적화를 지원합니다.

자주 묻는 질문

CSS 미니파이어란?

CSS에서 주석, 공백, 불필요한 문자를 제거해 파일 크기를 줄이는 도구입니다.

압축 후 스타일이 바뀌나요?

정상적인 압축은 스타일에 영향을 주지 않습니다.

0px를 0으로 바꾸는 이유는?

0에는 단위가 필요 없어 1~2바이트를 절약할 수 있습니다.

#ffffff를 #fff로 줄이는 방법은?

rrggbb 패턴이 rrgggbb로 반복되면 #rgb로 단축됩니다.

프로덕션에서 권장하는 도구는?

cssnano, clean-css, PostCSS 등을 권장합니다.

SCSS/LESS도 압축 가능한가요?

이 도구는 순수 CSS만 지원합니다. SCSS는 먼저 컴파일 후 사용하세요.

CSS 변수는 압축 시 문제없나요?

예, CSS 변수(--variable)는 정상적으로 처리됩니다.

미디어 쿼리도 압축되나요?

예, 미디어 쿼리 내부의 공백과 주석도 제거됩니다.
