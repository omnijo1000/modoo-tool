PDF 이미지 추출

PDF 각 페이지를 고해상도 PNG 이미지로 변환합니다. 해상도 배율을 선택하고 페이지별로 PNG를 개별 다운로드할 수 있습니다. 브라우저에서 완전 로컬 처리됩니다.

자주 묻는 질문

각 페이지를 개별 이미지로 받을 수 있나요?

예, 각 페이지 하단의 다운로드 버튼으로 PNG를 개별 저장합니다.

파일이 서버로 전송되나요?

아니요. 브라우저 내에서만 처리됩니다.

고해상도로 추출하려면?

해상도 배율을 3×으로 설정하면 216 DPI 품질로 추출됩니다.

페이지가 많으면 느린가요?

렌더링에 시간이 걸릴 수 있습니다. 특히 고배율·많은 페이지 시 느립니다.

PDF 내 이미지만 추출할 수 있나요?

현재는 페이지 전체를 PNG로 렌더링합니다. 내부 이미지 개별 추출은 지원하지 않습니다.

출력 형식은 PNG만 지원하나요?

현재 PNG 형식만 지원합니다.

암호화된 PDF도 처리되나요?

암호화되지 않은 PDF만 처리됩니다.

모바일에서도 사용 가능한가요?

예, 단 고해상도·많은 페이지는 모바일에서 느릴 수 있습니다.
