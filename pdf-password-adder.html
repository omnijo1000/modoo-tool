PDF 비밀번호 추가

PDF 파일에 사용자 비밀번호와 소유자 비밀번호를 설정합니다. 인쇄·복사·편집 권한을 세밀하게 제어할 수 있습니다. 모든 처리는 브라우저에서 로컬로 이루어집니다.

자주 묻는 질문

사용자 비밀번호와 소유자 비밀번호의 차이는?

사용자 비밀번호는 PDF를 열 때, 소유자 비밀번호는 편집/인쇄 권한 제어에 사용됩니다.

비밀번호 없이 권한만 제한할 수 있나요?

소유자 비밀번호만 설정하면 열람은 자유롭지만 권한은 제한됩니다.

파일이 서버로 전송되나요?

아니요. 브라우저 내에서만 처리됩니다.

어떤 암호화 방식을 사용하나요?

AES-128 암호화를 사용합니다.

설정한 비밀번호를 잊으면 어떻게 되나요?

비밀번호를 잃으면 복구가 매우 어렵습니다. 안전한 곳에 보관하세요.

비밀번호를 다시 제거하려면?

PDF 비밀번호 제거 도구를 사용하세요.

모바일에서도 사용할 수 있나요?

예, 모던 모바일 브라우저에서 동작합니다.

PDF 내용이 변경되나요?

아니요. 암호화만 추가되며 내용은 동일합니다.
