YAML to JSON 변환기

Kubernetes, Docker Compose, GitHub Actions 등 YAML 설정 파일을 JSON으로 즉시 변환합니다. API 연동, JSON 기반 도구 연계, 설정 데이터 처리 등에 활용할 수 있습니다. YAML 앵커·별칭·다중 문서를 지원합니다.

자주 묻는 질문

YAML 앵커(&)는 변환 시 어떻게 되나요?

앵커와 별칭은 변환 시 실제 값으로 해소되어 JSON에 인라인됩니다.

YAML 주석은?

JSON은 주석을 지원하지 않으므로 # 주석은 변환 시 제거됩니다.

다중 YAML 문서(---)는?

첫 번째 문서만 변환됩니다. 다중 문서는 각각 별도로 변환하세요.

YAML의 true/false/null은?

YAML 불리언(true/false, yes/no, on/off)과 null(~)은 JSON에서 동등한 값으로 변환됩니다.

큰따옴표·작은따옴표 문자열 차이는?

YAML에서 큰따옴표는 이스케이프 시퀀스를 처리하고, 작은따옴표는 리터럴로 처리합니다. 변환 후 JSON에서는 모두 일반 문자열이 됩니다.

Kubernetes YAML도 변환 가능한가요?

가능합니다. 단, Kubernetes YAML은 다중 문서(---)를 사용하는 경우가 많아 첫 번째 문서만 변환됩니다.

역변환(JSON→YAML)도 지원하나요?

json-to-yaml 도구를 이용하면 JSON을 YAML로 역변환할 수 있습니다.

탭 들여쓰기 YAML도 처리되나요?

YAML 사양은 들여쓰기에 탭을 허용하지 않습니다. 스페이스만 사용해야 합니다.
