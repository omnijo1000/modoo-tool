API 응답 뷰어

JSON, XML, 텍스트 형식의 API 응답을 붙여넣어 보기 좋게 포맷팅하고 구조를 분석합니다. Pretty Print, 트리 뷰, 상태 코드 해석 기능을 제공합니다.

자주 묻는 질문

어떤 응답 형식을 지원하나요?

JSON, XML, 일반 텍스트를 지원합니다. 형식을 자동으로 감지합니다.

트리 뷰는 어떻게 작동하나요?

JSON 객체의 계층 구조를 트리 형태로 표시합니다. 노드를 클릭하여 접고 펼 수 있습니다.

HTTP 상태 코드를 해석해주나요?

예, 200 OK, 404 Not Found 등 일반적인 상태 코드의 의미를 표시합니다.

API 요청을 직접 보낼 수 있나요?

이 도구는 응답 분석에 초점을 맞춥니다. 실제 API 요청은 API Tester를 이용하세요.

큰 JSON 응답도 처리할 수 있나요?

브라우저 메모리 내에서 처리하므로 매우 큰 응답(수 MB 이상)은 느릴 수 있습니다.

포맷팅된 결과를 복사할 수 있나요?

예, "포맷 복사" 버튼으로 들여쓰기된 JSON을 클립보드에 복사합니다.

응답 헤더도 분석할 수 있나요?

현재는 바디 분석에 집중합니다. 상태 코드는 별도로 입력할 수 있습니다.

서버로 데이터가 전송되나요?

아니오, 모든 처리는 브라우저에서만 이루어집니다.
