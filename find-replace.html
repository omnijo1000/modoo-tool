찾기 & 바꾸기 도구 사용법

텍스트에서 특정 단어나 패턴을 검색하여 다른 텍스트로 대체합니다. 대소문자 구분, 정규식 모드, 전체 단어 일치 옵션을 지원하며, 모든 처리는 브라우저 내에서 이루어집니다.

정규식(Regex) 모드란 무엇인가요?

정규식은 텍스트 패턴을 표현하는 언어입니다. 예: \d+는 하나 이상의 숫자, \s+는 공백 문자에 일치합니다. 정규식 모드 활성화 시 찾기 필드에 정규식 패턴을 입력할 수 있습니다.

전체 단어 옵션은 어떻게 작동하나요?

"cat"을 전체 단어로 검색하면 "cats"나 "catch"는 매칭되지 않고 독립된 단어 "cat"만 매칭됩니다. 변수명이나 특정 단어를 정확히 교체할 때 유용합니다.

대소문자 구분 옵션은 어떤 역할을 하나요?

활성화하면 "Hello"와 "hello"를 다른 단어로 취급합니다. 비활성화(기본값)하면 대소문자에 관계없이 모두 매칭합니다.

일부 항목만 선택해서 교체할 수 있나요?

현재 도구는 일치하는 모든 항목을 한 번에 교체합니다. 특정 항목만 교체하려면 교체 후 Ctrl+Z로 되돌리거나, 텍스트 에디터의 단계별 찾기/바꾸기 기능을 활용하세요.

입력한 텍스트가 서버에 저장되나요?

아니요. 모든 처리는 브라우저 내에서 이루어지며, 입력한 텍스트는 서버로 전송되거나 저장되지 않습니다. 민감한 내용도 안심하고 사용할 수 있습니다.

여러 개의 패턴을 동시에 교체할 수 있나요?

현재는 한 번에 하나의 패턴만 교체할 수 있습니다. 여러 패턴이 필요하다면 교체를 순차적으로 반복해서 적용하면 됩니다. 자동화가 필요하면 스크립트 언어(Python, JavaScript)의 replace 함수를 활용하세요.

줄바꿈 문자(개행)를 찾아 교체할 수 있나요?

정규식 모드를 켠 뒤 을 찾기에 입력하면 줄바꿈 문자를 찾을 수 있습니다. 교체값에도 을 사용해 줄바꿈을 삽입할 수 있습니다.

백레퍼런스($1)를 사용해 일부를 보존하며 교체할 수 있나요?

정규식 모드에서 교체값에 $1, $2 등의 백레퍼런스를 사용하면 캡처 그룹을 유지하며 교체할 수 있습니다. 예: 찾기 (d{4})-(d{2}), 교체 $2/$1 → 날짜 형식 변환.
