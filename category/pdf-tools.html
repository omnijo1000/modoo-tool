PDF Tools

Free online tools for merging, splitting, and converting PDF files — no uploads needed

PDF manipulation used to require expensive software or uploading sensitive documents to cloud services. MODOO HUB's PDF tools eliminate both problems: everything runs locally in your browser using PDF.js and PDF-lib, which means your documents never leave your device.

The PDF Merge tool combines up to 20 PDF files into one, with drag-and-drop reordering before you merge. The PDF Split tool extracts specific pages, splits by every N pages, or separates every page into its own file. The PDF to Image converter renders each page as a high-quality PNG or JPEG — useful for creating image previews of documents or extracting specific pages as visuals. Image to PDF goes the other direction, combining multiple images into a single PDF in seconds.

No watermarks, no file size limits imposed by a free tier, no sign-up. Just private, local PDF processing.

Frequently Asked Questions
How many PDFs can I merge at once?▼
You can merge up to 20 PDF files in a single operation. Drag to reorder pages before merging. All processing happens in your browser using PDF-lib — no files are uploaded to any server.
Can I split specific page ranges from a PDF?▼
Yes. The PDF Splitter lets you extract specific page ranges, split by every N pages, or split each page into a separate individual PDF file.
What image format does PDF to Image output?▼
The tool outputs PNG by default (best for documents with text and graphics) or JPEG for smaller file sizes. Each PDF page is converted to a separate image file that you can download individually or as a zip.
Is it safe to process confidential PDFs here?▼
Yes. All PDF processing runs entirely in your browser using PDF.js and PDF-lib — your documents are never uploaded to any server or stored anywhere. Completely safe for confidential documents.
