중복 줄 제거기란?

텍스트의 중복된 줄을 자동으로 제거하고 고유한 줄만 남깁니다. 대소문자 구분·공백 처리·빈 줄 제거·결과 정렬 옵션을 지원하며, 입력 즉시 실시간으로 처리됩니다.

대소문자 구분 옵션은 어떻게 동작하나요?

대소문자 구분 활성화 시 "Apple"과 "apple"을 서로 다른 줄로 취급합니다. 비활성화 시 대소문자 관계없이 같은 내용으로 판단하여 하나만 남깁니다. 목록 정리에는 비활성화가 더 유용한 경우가 많습니다.

앞뒤 공백 제거 옵션은 무엇인가요?

각 줄의 앞뒤 공백(스페이스, 탭)을 제거한 후 중복 여부를 비교합니다. 예: " apple "과 "apple"을 같은 줄로 처리합니다. 데이터 정리나 CSV 처리 시 유용합니다.

결과 정렬 옵션을 사용하면 원래 순서가 바뀌나요?

네. 정렬 옵션 활성화 시 결과가 알파벳순으로 정렬됩니다. 순서가 중요한 경우에는 비활성화하세요 — 기본적으로 첫 등장 순서를 유지합니다.

어떤 경우에 이 도구를 사용하나요?

이메일 목록 정리, 키워드 목록 중복 제거, 로그 파일 분석, CSV 데이터 전처리, 코드에서 중복 import 제거 등 다양한 상황에 유용합니다. 대용량 텍스트도 브라우저에서 빠르게 처리합니다.

처리할 수 있는 텍스트 크기 제한이 있나요?

엄격한 제한은 없지만, 매우 큰 텍스트(수십만 줄 이상)는 처리 속도가 느려질 수 있습니다. 일반적인 데이터 정리 작업(수천~수만 줄)에서는 빠르게 동작합니다.

중복 제거 후 줄 수가 몇 개 줄었는지 알 수 있나요?

처리 완료 후 원본 줄 수, 제거된 줄 수, 남은 줄 수가 통계로 표시됩니다. 데이터 정제 전후의 변화를 쉽게 파악할 수 있습니다.

이 도구로 CSV의 중복 행을 제거할 수 있나요?

가능합니다. CSV 파일의 각 행을 줄 단위로 비교해 중복 행을 제거합니다. 단, 헤더 행이 중복 제거 대상이 되지 않도록 주의해서 붙여넣으세요.

첫 번째 등장한 줄만 남기나요, 아니면 마지막 등장한 줄만 남기나요?

기본적으로 처음 등장한 줄(first occurrence)을 유지하고 이후 중복 줄을 제거합니다. 원본 순서를 최대한 보존하는 방식입니다.
